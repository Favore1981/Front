Команди, які були використані під час виконання домашнього завдання

для підключення репозиторію та перший коміт на гілку main

  • git init
  • git add .
  • git commit -m "name commit"
  • git branch -m main
  • git remote add origin https://github.com/Favore1981/Front.git
  • git push -u origin main

використовується команда git status постійно

для видалення файлів з гілки main використував ці команди

  • git rm --cached -r назва_файлу
  • git commit -m "remove назва from repository"
  • git push origin main

спочатку моя головна гілка називалась master тому я змінив її назву командою git branch -m main

далі створив від гілки main гілку за допомогою команди git checkout -b lesson12 і працював з цією гілкою в терміналі. Закомітив проект до відгалуженої гілки, у гитхаб зробив pull request та зробив merge на гілку main
